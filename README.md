# Popayán Go - Proyecto React

Proyecto React moderno configurado con **Vite** y **React 19**.

## 🚀 Instalación y Configuración

El proyecto ya está configurado con todas las dependencias instaladas. Solo necesitas iniciar el servidor de desarrollo:

### Iniciar el servidor de desarrollo
```bash
npm run dev
```

Esto abrirá automáticamente la aplicación en: `http://localhost:5173/`

### Compilar para producción
```bash
npm run build
```

Los archivos compilados se guardarán en la carpeta `dist/`.

### Previsualizar la compilación
```bash
npm run preview
```

### Verificar código con ESLint
```bash
npm run lint
```

## 📁 Estructura del Proyecto

```
src/
├── App.jsx          # Componente principal
├── App.css          # Estilos de la aplicación
├── index.css        # Estilos globales
├── main.jsx         # Punto de entrada
└── assets/          # Archivos estáticos

public/              # Archivos públicos
index.html          # HTML principal
vite.config.js      # Configuración de Vite
package.json        # Dependencias del proyecto
```

## 📦 Dependencias Principales

- **React 19.2.6**: Librería de interfaz de usuario
- **React DOM 19.2.6**: Renderización en el DOM
- **Vite 8.0.12**: Herramienta de compilación rápida
- **ESLint**: Verificación de código

## ✨ Características

- ✅ HMR (Hot Module Replacement) - Recarga en caliente

- ✅ Configuración de React optimizada
